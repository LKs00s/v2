🔧 Analizador de Cotizaciones - Embedded

Ejemplo de cómo integrar el analizador en tu sitio web

📋 Cómo integrar en tu sitio web

🎯 Código básico para iframe:

<iframe src="https://frolicking-fox-f0ca05.netlify.app" width="100%" height="800" frameborder="0" title="Analizador de Cotizaciones"> </iframe>

📐 Tamaños predefinidos:

💡 Recomendaciones:

  • Altura mínima: 600px para funcionalidad completa
  • Ancho: 100% para diseño responsivo
  • Carga: Usa loading="lazy" para mejor rendimiento
  • Accesibilidad: Incluye siempre el atributo title

🎨 Código responsivo avanzado:

<div style="position: relative; width: 100%; height: 0; padding-bottom: 56.25%;"> <iframe src="https://frolicking-fox-f0ca05.netlify.app" style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;" frameborder="0" title="Analizador de Cotizaciones" loading="lazy"> </iframe> </div>

⚙️ Integración con JavaScript:

// Crear iframe dinámicamente const iframe = document.createElement('iframe'); iframe.src = 'https://frolicking-fox-f0ca05.netlify.app'; iframe.width = '100%'; iframe.height = '800'; iframe.frameBorder = '0'; iframe.title = 'Analizador de Cotizaciones'; iframe.loading = 'lazy'; // Agregar al contenedor document.getElementById('cotizaciones-container').appendChild(iframe);
